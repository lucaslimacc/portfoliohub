01 · Sobre

Quem sou eu

Sou Lucas Lima, estudante de Análise e Ciência de Dados no Centro Universitário de Brasília (CEUB). Meu foco está na interseção entre análise exploratória, visualização de dados e modelos de Machine Learning aplicados a problemas reais.

Trabalho com Python e ecossistema científico (Pandas, NumPy, Scikit-learn), SQL para consultas e manipulação de dados, e Jupyter Notebooks para documentar e apresentar análises de forma reproduzível.

Este portfólio reúne os projetos desenvolvidos ao longo do curso — desde atividades de Álgebra Linear aplicada até análises completas com Pandas — versionados e organizados no GitHub.
